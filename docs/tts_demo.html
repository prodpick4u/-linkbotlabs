🔑 API Key Setup

Amazon API Key:

  1. Sign up at Amazon Affiliate Program.
  2. Access Product Advertising API keys in your account settings.
  3. Copy keys into Prodpick4u desktop app to enable full search.

YouTube API Key:

  1. Go to Google Cloud Console.
  2. Create a new project → enable YouTube Data API v3.
  3. Generate API key and paste into Prodpick4u desktop app.
