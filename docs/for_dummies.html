1️⃣ Registering for Amazon Affiliate

  1. Go to Amazon Associates.
  2. Sign Up → Fill account info → Submit → Get Affiliate Tag (mytag-20).

2️⃣ Getting Amazon Product API Keys

  1. Amazon Associates → Tools → Product Advertising API → Manage Credentials.
  2. Create credentials → Save Access Key ID & Secret Access Key.

3️⃣ Obtaining YouTube API Key

  1. Google Cloud Console → Create Project → Enable YouTube Data API v3.
  2. Create API Key → Copy for LinkBotLabs usage.
